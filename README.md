# 🚨 Police & Thief Simulation

멀티스레드 기반 경찰-도둑-금고 시뮬레이션 프로젝트입니다.
여러 스레드(도둑/경찰/리포터)가 동시에 동작하면서 공유 자원(금고)에 접근할 때 발생할 수 있는 동시성 문제를 직접 제어하고, 
진행 상황을 콘솔/GUI로 시각화합니다.

---

## 구성 요소

### 💰 금고
- 위치: (10, 10) 고정
- 초기 잔액: 100,000원
- 역할:
  - 도둑이 훔친만큼 총액에서 감소
  - 도둑이 체포되면 금액 복구

---

### 🥷 도둑
- 여러 명 존재
- 일정 주기로 랜덤 이동
- 금고 위치에 도착하면 1000 ~ 10000원 랜덤 금액 훔치기 시도
- 체포 시 활동 중지

---

### 👮🏻‍♂️ 경찰
- 여러 명 존재
- 자신의 구역 내에서 활동
- 구역 내 가장 가까운 도둑을 찾고 추적
- 체포 시 도둑이 훔친 금액 회수 - 같은 위치가 되면 도둑 체포 및 체포 횟수 누적

---

### 💸 리포터
- 일정 주기마다 상태 출력
- 출력 정보:
  - 금고 잔액
  - 총 도난액
  - 총 회수액
  - 체포 횟수

---

### Swing GUI
- 콘솔에서 보던 내용을 GUI로 시각화 
- 출력 정보는 리포터와 동일

---

## ⚙ 기술 스택

- Language : Java 17
- Build / Packaging
  - Maven
  - Eclipse Runnable JAR Export
- GUI : Java Swing, JPanel Custom Painting(Graphics2D)
